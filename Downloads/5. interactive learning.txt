## 📘 Konsep Dasar

Formula elastisitas permintaan:
\[
E_d = \frac{\%\Delta Q_d}{\%\Delta P}
\]


```{r, context="server"}
quiz(
  question("Jika harga naik 10% dan permintaan turun 20%, berapakah elastisitas permintaannya?",
    answer("-0.2"),
    answer("-2", correct = TRUE),
    answer("2"),
    answer("0.5"),
    allow_retry = TRUE,
    random_answer_order = TRUE
  )
)

```


📘 **Penjelasan:**
- `quiz()` dan `question()` → fungsi dari `learnr` untuk membuat kuis.  
- `context="server"` → penting! artinya bagian ini dijalankan dalam *server environment* Shiny.
